Le language HTML

Langage de balisage utilisé pour la création de pages web, permettant notamment de définir des liens hypertextes.

Le language CSS

Le CSS est un langage informatique utilisé sur l'internet pour mettre en forme les fichiers HTML. Elle permet de gérer le design des pages HTML.

Le language Java Script

Le JavaScript est un langage informatique utilisé sur les pages web pour ajouter de petites animations ou effets.
